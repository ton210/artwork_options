# MunchMakers B2B Landing Pages - Designer Guide

## Overview
This document contains all 14 B2B landing pages that need image updates. Each page targets a specific cannabis industry segment with unique messaging and visuals.

## How to Access and Edit Pages

### BigCommerce Admin Access
1. Log into BigCommerce Admin: https://store-tqjrceegho.mybigcommerce.com/
2. Navigate to: **Storefront > Web Pages**
3. Search for each page by its ID number or title
4. Click "Edit" to modify the page
5. Update images directly in the WYSIWYG editor

### Important Notes for Designers
- All pages currently have placeholder image spots marked in the HTML
- Images should be high-quality, professional, and relevant to each industry segment
- Recommended image dimensions: 1200x800px for hero images, 600x400px for feature images
- File format: JPG or PNG, optimized for web (under 200KB per image)
- Upload images through BigCommerce Media Manager for best performance

- **Color Space:** sRGB
- **Resolution:** 72 DPI for web
- **Naming Convention:** page-name_image-type_v1.jpg
